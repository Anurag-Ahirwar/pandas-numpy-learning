# NumPy and Pandas Learning Repository

Welcome to my personal learning repository for NumPy and Pandas! This repo contains code examples, notes, exercises, and projects as I build my skills in data manipulation and analysis using Python.

## 📚 What I'm Learning

- **NumPy**: Arrays, mathematical operations, broadcasting, and more
- **Pandas**: DataFrames, Series, data cleaning, analysis, and visualization

## 🚀 Getting Started

### Prerequisites
- Python 3.7+
- Git

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/Anurag-Ahirwar/pandas-numpy-learning.git
   cd pandas-numpy-learning
   ```

2. Install required packages:
   ```bash
   pip install numpy pandas matplotlib jupyter
   ```
